Advanced Python: Building Scalable Applications

Recommended duration: 4 modules (4 days @ 4 hours per day)

Course Description:

This training program provides insight on leveraging various python modules, libraries and tools for developing scalable applications using Python.

Target Audience (who should attend):

  • Engineers who wish to develop large-scale python applications

Course Track: Advanced

Pre-requisites:

  • Participants should be comfortable with the following technologies:
    • Good understanding of python programming that include:
      • variables and datatypes
      • flow control statements
      • comprehensions and file I/O
      • built-in data types (string, list, tuple, set, dict)
      • functions, modules, namespaces and OOP
      • Decorators, generators and yield semantics

Course Objectives:

  • Learn how to create multitasking/parallel processing applications in Python leveraging process and threads and understand their intricacies.
  • Learn how to manage thread synchronization and IPCs using pythonic approach.
  • Learn how to leverage serialization and data-persistence modules in python.
  • Learn how to parse and process various file formats (XML, JSON, YAML, INI).
  • Learn how to create network servers and clients.
  • Learn how to interface with SQL-compliant and NoSQL databases.
  • Learn how to implement RESTful API and expose them.

What will NOT be covered:

  • Basic concepts and idioms in Python (participants are expected to know the same).
  • In-depth coverage on web application frameworks (due to schedule-constraints).
  • GUI programming.
  • Python extension libraries/modules.
  • Data analytics/ML/Text processing related concepts

Training mode:

  • This training program is designed to be demonstration oriented.

  • Most concepts will be explained via demonstrating code and participants are expected to learn by practicing the same.

Lab requirements:

  • As the training is highly lab oriented, each participant attending the training program must be provided with a computer/laptop with the following software installed:
    • Python 3.10+ installed for Windows/Linux/MacOS.
    • VisualStudio Code with Python extensions enabled.

Course Agenda

Module 1

Threads, Processes and Coroutines

  • Concurrency Vs Parallelism: Choosing generator/coroutines Vs Threads/Processes
  • Introduction to threads and processes.
  • Python threading module API
    • Creating and managing threads.
    • An overview on threading module.
    • Using the Thread class and the Timer class.
    • Active threads Vs. Daemon threads.
    • Helper functions in the threading module.
  • Python multiprocessing module API
    • Multitasking using multiprocessing.Process
    • Process Vs Thread: performance and design implications.
    • Similarities and differences between Process and Thread class API
    • Helper functions in the multiprocessing module.
  • Creating thread-pools and process-pools using concurrent.futures package
  • Using gevent framework for massive I/O concurrency work-loads
  • Implementing co-routines using Python built-in async support
  • An overview on uvloop and trio
  • Best architectural design practices while choosing between different concurrency models.

Module 2

Thread / Process synchronization mechanisms

  • Mutual exclusion patterns using Lock and RLock.
  • Wait/notify patterns using Condition and Event.
  • Synchronizing flow-control using Barrier.
  • Bandwidth management/control using Semaphore and BoundedSemaphore.
  • Implementing Producer/Consumer patterns using Queue.
  • Creating thread-local objects using threading.local().
  • Sharing/Exchanging data between processes
    • Streaming data using Pipe and Queue
    • Sharing counters and buffers using Value and Array
    • Sharing python lists and dictionaries using Manager
    • Creating and managing shared memory using multiprocessing.shared_memory features

Module 3

Network socket API

  • An overview of the socket module
  • Creating network servers and clients using socket module
  • Using gevent.sockets for concurrent socket connection handling
  • Using asyncio for concurrent socket programming

Marshalling and Data Persistence

  • An overview on pickle, dill and shelve modules
  • Parsing common file formats: CSV, JSON, YAML, TOML and MsgPack
  • Parsing HTML and XML using lxml

Profiling Python Performance and Memory Usage

  • Using cProfile and timeit module
  • Using sys.getsizeof(), sys.getrefcount(), sys.getswitchinterval()`
  • Using the line_profiler and Memray

Module 4

Connecting to databases

  • An overview on DBAPI
    • De-coupling database driver operations from high-level persistence API
    • Use-case examples using SQLite3/MariaDB/PostgreSQL
  • Working with NoSQL databases
  • Python API for redis and MongoDB (overview)

Creating Web APIs

  • An overview on RESTful API architecture
  • Consuming web API using python requests and httpx
  • An overview on WSGI and ASGI
  • WSGI / ASGI servers: gunicorn and uvicorn
  • An overview on starlette ASGI framework
  • Creating and exposing web API using FastAPI framework
  • Discovering web APIs using /docs (Open-API) and /redocs interface
  • Creating validating models using pydantic
  • Security Considerations: Authentication and Authorization techniques

Odds and Ends

  • Building Scalable Web Applications: Best architectural practices
    • Case studies and examples of common web application architecture
    • Containerization and Cloud Deployment of web applications

Q & A
